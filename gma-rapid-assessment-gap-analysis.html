  • Trade transaction framework (Slides 45–46) — 3 import regimes (Freedom, Approval, Limitation). Grains = Category I (Freedom Arrangement). WAEMU CET: 5% + statistical duty 1% + AU levy 0.2% + community levies 1.3%.

  • Payment system architecture (Slide 9) — STAR-UEMOA (RTGS, ≥XOF 50M, real-time, cutoff 5:00 PM), SICA-UEMOA (domestic funds transfer,
  • BCEAO batch rejection risk (Slide 22) — If any payment in a submitted FX batch is incorrect, the entire batch gets rejected. When liquidity is tight, BCEAO finds more reasons to reject.
